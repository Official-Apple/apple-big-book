# 模块 5-01 数列定义  

## 1.数列的定义  
按一定次序排列的一列数称为数列  
一般形式：a_1，a_2，a_3，...，a_n，...，简记为{a_n}   
`注意` 它可以理解为以正整数集 (或它的有限子集) 为定义域的函数.   
运用函数的观念分析和解决有关数列问题，是一条基本思路. 递推是数列特有的表示法，
它更能反映数列的特征。   

数列的表示形式：  
1. 列举法  
   1，3，5，7  
2. 通项法  
   a_n=2n-1   
   - n=1 => 2·1-1=1  
   - n=2 => 2·2-1=3   
   - n=3 => 2·3-1=5  
   - n=4 => 2·4-1=7  
3. 递推法  
   前后项之间的关系  
   `a_(n+2)=a_(n+1)+a_n`   
   1，3，4，7，11，18，...    
   - 1+3=4  
   - 3+4=7  
   - 4+7=11  
   - 7+11=18    
    

## 2.通项公式   
`a_n=f(n)` (第n项a_n与项数n之间的函数关系)   
`注意` 并非每一个数列都可以写出通项公式（例如数列变化不是特别明显的数列不能使用通项公式），有些数列的通项公式也并非是唯一的。    

通项公式并非唯一的`a_n=f(n)`形式是可以灵活变化：  
a_n=3n-1=f(n)   
`a_n=3n-1`这个通向也可以看作是一个`f(n)`函数   
=> a_1=3·1-1=2  
=> a_2=3·2-1=5  

已知通项公式，就能快速的求出函数中任何一项数值  
比如求`a_8`  
=> a_8=3·8-1=23   
这样就求出了值为`23`  


## 3.数列的前n项和  
数列的前n项和标记`S_n=a_1+a_2+a_3+...+a_n`   
(必须从第一个开始`a_1`一直到当前项的和)   
- n=1 => S_1=a_1    
- n=2 => S_2=a_1+a_2     
- n=7   
  => S_7=a_1+a_2+...+a_6+a_7   
  => a_1+a_2+...+a_6=S_6     
  => S_7=S_6+a_7   

即`S_7=S_6+a_7`等价与下面这个公式:   
S_n=S_(n-1)+a_n   <=>  S_n-S_(n-1)=a_n   

`a`代表个体  
`S`代表全体  
S_7=S_5+a_6+a_7    
`S_5`：全体即集合    
`a_6跟a_7`：两个单独的个体   

`注意`这里需要注意`n`下标不为0的情况，如果`n-1`代表这个`S`是没有任何意义的 
就跟编程中的空集合类似，所以n必须(n≥2)    


## 4.a_n与S_n的关系（重要）  
∑=sigma(西格玛)表示求和运算

1. 已知a_n，求S_n  
   公式：`S_n=a_1+a_2+...+a_n=∑_(i=1)^n(a_i)`  

2. 已知S_n，求a_n   
   公式：`a_n=[n=1 -> a_1=S_1, n≥2 -> S_n-S_(n-1)]`   
   `注意` n不能等于1所以n=1需要分段讨论   
   `n=1 => a_1=S_1`   
   `n≥2 => S_n-S_(n-1)`    
    用这S_n-S_(n-1)两个的差就表示a_n   
    即`S_n-S_(n-1)=a_n`  
    此外在有些特殊的情况下 a_1=S_1 满足 S_n-S_(n-1) 的结果
    就可以直接用 S_n-S_(n-1) 表示不用在分段讨论了如果a_1的值不满足
    S_n-S_(n-1)的结果还是需要进行分段讨论   


##  考点1.已知S_n，求a_n  
根据公式：`a_n=[n=1 -> a_1=S_1, n≥2 -> S_n-S_(n-1)]`  来求解分析   
使用S的差值就可以求出a_n

###### ［例1］已知数列{a_n}的前n项和S_n=n^2+1，则a_5=?  
A `9`  
B `8`  
C `6`  
D `4`  
E `16`  

n=5   
a_5=S_5-S_4 
代入`S_n=n^2+1`  
=> (5^2+1)-(4^2+1)  
后面的常数项1可用约掉不影响结果  
=> 25-16  
=> 9 

S_n常数项`S_n=n^2+1`的+1影响的是a_1的值，`注意`不是S_n的 [n≥2 => S_n-S_(n-1)] 公式的-1项    
n=1   
a_1=S_1=1+1=2   
原本的a_1=1，如果考虑常数项可能会得到S_1=2   
`S_n中的常数项只影响a_1不影响其他值`   

答案为A  



###### ［例2］已知数列{a_n}的前n项和S_n满足log_2(S_n+1)=n+1,则a_6=?  
A `32`  
B `12`  
C `64`  
D `72`  
E `24` 

根据对数的定义：  
log_b(a)=c  即 b^c=a   
转换 `b^c=a`    
S_n+1=2^(n+1)   
去除+1即减去1出S_n  
S_n=2^(n+1)-1  
这里是a_6不是a_1直接代入`n≥2 => S_n-S_(n-1)`公式   
a_6=S_6-S_5  
这里不是a_1可以直接约掉后面的1即`2^(n+1)`  
=> S_6=2^(6+1)    
=> S_5=2^(5+1)  
=> 2^7-2^6  
因式分解计算次方
=> 2^7=2^6·2^1    
=> 2^6=2^6·1   
两项中存在`2^6`共同因子   
=> 2^6(2^1-1)     
=> 2^6·1  
=> 64  

补充指数法则: `a^m+n=a^m·a^n`    
2^(n+1)-2^n=2^n   
=> 2^1·2^n-2^n   
=> 2·2^n-2^n  
=> 2·2^n-1·2^n  
=> 2^n   

答案为C   


###### ［例3］已知数列 {a_n} 的前n项和S_n=n^3，则a_6+a_7+a_8+a_9+a_10=?  
A `725`  
B `775`  
C `825`  
D `855`  
E `875`  

a_6+a_7+...+a_10   
可以看作a_10-a_5即减去a_6前面的数就代表 `a_6+a_7+...+a_10` 的结果  
a_6+a_7+...+a_10=S_10-S_5   
代入S_n=n^3  
=> 10^3-5^3  
=> 10^3=10·10·10=1000   
=> 5^3=5·5·5=125  
=> 1000-125  
=> 875  

a_m+a_(m+1)+...+a_k(最终项)=S_k-S_m-1  
必要条件(k>m)  

答案为E  



###### ［例4］已知数列{a_n}的前n项和S_n=(10^n)-1，求{a_n}的通项公式？     

基本通项公式：   
a_n=[n=1 -> a_1=S_1, n≥2 -> S_n-S_(n-1)]   

1. n=1:    
   a_1=S_1=10^1-1   
   => 10-1  
   => 9 
    
2. n≥2:  
   a_n=S_n-S_(n-1)    
   两常数项相减或相加可以忽略因为结果最终一致 即`S_n=(10^n)-1`=`S_n=10^n`     
   代入`S_n-S_(n-1)`验证：      
   S_n=(10^n)-1      
   S_(n-1)=(10^n-1)-1      
   => ((10^n)-1)-((10^n-1)-1)      
   `第二项乘括号前的符号-1为1`      
   => (10^n)-1-(10^n-1)+1        
   `常熟项1相互抵消`          
   => (10^n)-10^(n-1)      
   拆解10^n的次方：     
   => 10^n = 10·10^(n-1)   
   提取公因式`10^(n-1)`    
   => 10^(n-1)·(10-1)   
   => 10^(n-1)·9  
   => 9·10^(n-1)  

验证n≥2是否等于n=1计算出来的9，如果不等于需要进行分开写(分段讨论)    
n=1：  
=> 9·10^(1-1)  
=> 9·10^0  
=> 9·1  
=> 9   

综上即把两个`[n=1 -> a_1=S_1, n≥2 -> S_n-S_(n-1)]`公式和在一起写：     
a_n=9·10^(n-1)  

`注意` 如果`S_n=(10^n)-1`后面的常数项不是`-1`会影响首项，不影响其他项 `［例1］题目有详细说明`     
即如果不是`-1`需要把两个公式分开写不能合在一起，因为n≥2是不会等于n=1的结果的，所以合在一起后还需要验证

##  考点2.已知a_n，求S_n   
采用公式：S_n=a_1+a_2+...+a_n=∑_(i=1)^n(a_i) 求解，结合对通项裂项
进而采用相消求和法. 这是分解与组合思想在数列求和中的具体应用，裂项法的实质是将数列中的每项
（通项）分解，然后重新组合，使之能消去一些项最终达到求和的目的.

###### ［例5］在数列{a_n}中，a_n=n/2，又b_n=2/(a_n·a_(n+1))，则数列{b_n}的前99项和为？  
A `99/25`  
B `101/25`  
C `202/25`  
D `198/25`  
E `298/25`  

已知`a_n=n/2`计算`b_n`:  
b_n=2/(a_n·a_(n+1))
=> 2/(n/2·(n+1)/2)  
展开分母   
=> n/2·(n+1)/2 = (n(n+1))/4  
=> 2/((n(n+1))/4)  
计算除分数时候直接反转分母然后进行乘即可  
=> 2·(4/(n(n+1)))    
=> 8/(n(n+1))    
`1/(n(n+k))分式裂项公式`：  
`1/(n(n+k))=(1/k)((1/n)-(1/(n+k)))`     
`这里需要注意k的情况`    
=> 8/(n(n+1))=(8/1)((1/n)-(1/n+1))   
=> 8((1/n)-(1/(n+1)))     

1. b_n=8((1/n)-(1/(n+1)))   
2. S_99=b_1+b_2+...+b_99    
   将每一项代入`8((1/n)-(1/(n+1)))`        
   每一项都有8直接提取8      
   => 8[(1/1)-(1/2)+(1/2)-(1/3)+...+(1/99)-(1/100)]     
   这里的每一项之间都可以进行抵消     
   比如`-(1/2)+(1/2)`,`-(1/3)+(1/3)`以此类推都可以进行合并同类项相互抵消    
   => [(1/1)-(1/2)+(1/2)-(1/3)+...+(1/99)-(1/100)]=((1/1)-(1/100))       
   => 8(1-(1/100))    
   => 1-(1/100) = (100/100)-(1/100) = 99/100   
   => 8·(99/100)   
   => (8/1)·(99/100)   
   => 792/100   
   简化分数792跟100之间最大公约数(同时整除多个整数的最大正整数)为4     
   => 792÷4/100÷4  
   => 198/25

###### ［例6］数列{a_n}的通项公式是 a_n=1/((√n)+(√n+1)) 若前n项的和为10，则项数n=？  
A `119`  
B `120`  
C `121`

有理化分母运算规则: 将含有根号或虚数的分数的分母转化为无根号或实数形式的过程，
这一过程通常涉及将分子和分母同时乘以一个分母共轭(类似于某种结构或对象的`镜像`或`反转`操作)的数， 
以消除分母中的根号或虚部，从而简化表达式。    

分母共轭概念（适用与分母中的根号或虚部）：  
当分母中有一个如`a+b`或`a-b`的二项式时， 其共轭分别是`a-b`和`a+b`， 
例如对于`√n+√m`其共轭形式为`√n-√m`，即取反      

1. 分母有理化消除平方根号       
   1/((√n)+(√n+1))    
   分母转换为分母共轭数      
   => (√n)+(√n+1) = (√n+1)-(√n)     
   乘以分母的共轭形式消除平方根    
   => (1/(√n+(√n+1)))·(((√n+1)-√n)/((√n+1)-√n))     
   => ((√n+1)-√n)/(√n+(√n+1))·((√n+1)-√n)       
   转换平方差公式`a^2-b^2=(a+b)(a-b)`    
   => (√n+(√n+1))·((√n+1)-√n) = (√n+1)^2-(√n)^2    
   => (√n+1)^2-(√n)^2 = n+1-n   
   合并同类项   
   => n-n+1    
   => 1    
   即简化后分母为1   
   => (√n+1)-√n)/1 = √n+1-√n   
2. 求项数n的值         
   1. a_n=√n+1-√n      
   2. S_n=((√1+1)-√1)+((√2+1)-√2)+...+((√n+1)-√n)      
      => (√2-√1)+(√3-√2)+...+((√n+1)-√n)     
      这里每一项的前项跟下一项的后项之间可以进行抵消        
      比如`√2-√2`,`√3-√3`以此类推都可以进行合并同类项相互抵消    
      => (√2-√1)+(√3-√2)+...+((√n+1)-√n) = (√n+1)-√1   
      => (√n+1)-1    
      => (√n+1)-1 = 10   
      移项            
      => (√n+1)=11   
      去掉根号两边平方   
      => (√n+1)^2=11^2     
      => n+1=121   
      => n=120
3. 知识提炼     
   1. `1/((√n)+(√n+k))分式裂项公式`：    
      `1/((√n)+(√n+k))=(1/k)((√n+k)-√n)`         
      => k=1: 1/((√n)+(√n+1))=(1/1)((√n+1)-√n)      
      =====> 1/(√n+(√n+1))=(√n+1)-√n        
      => k=2: 1/((√n)+(√n+2))=(1/2)((√n+2)-√n)        
      =====> 1/(√n+(√n+2))=(1/2)((√n+2)-√n

答案为B 


###### ［例7］S_99=(1/2!)+(2/3!)+(3/4!)+...+(99/100!)的值为？

      这里每一项的前项跟下一项的后项之间可以进行抵消        
      比如`2!-2!`,`3!-3!`以此类推都可以进行合并同类项相互抵消    
      => (2!-1!)+(3!-2!)+...+(100!-99!) = 100!-1!    
      => 100!-1!
