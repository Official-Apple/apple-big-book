# 模块4-03 一次不等式（组）

## 1. 不等式的定义
用不等号连接的两个（或两个以上）解析式称为不等式，使不等式成立的未知数的取值称为不等式的解 
（不等号包括`＞`、`<`、`≤`、`≥`、`≠` 五种）

## 2. 不等式的分类
按照不等式的解的情况可以将不等式分以下三类：
- （1）绝对不等式；解集为R的不等式；  `x^2+2>0`
- （2）条件不等式：解集实数集的非空真子集的不等式；  `x^2-2>0`
- （3）矛盾不等式：解集空集的不等式。 `x^2+2<0 没有实数解`

###### ［例15］若1/a<1/b<0，给出下列不等式：(1) 1/a+b<1/ab; (2) |a|+b>0; (3) a-1/a>b-1/b; (4) lna^2>lnb^2 其中正确的不等式是？  
A `(1)(4)`  
B `(2)(3)`  
C `(1)(3)`  
D `(2)(4)`  
E `(1)(2)`  

特值:  a=-2  b=-3  
一般推导方法: 1/a<1/b<0  <=>  b<a<0  
(1) 1/a+b<0<1/ab 成立(a+b负数,a乘b正数)  
(2) 换算绝对值|b|>|a|，b<0，所以成-b>|a|, 移项|a|+b<0 未成立  
(3) 从题干得出1/a<1/b  
&nbsp;&nbsp; 两边乘-1符合反转   -(1/a)>-(1/b)  
&nbsp;&nbsp; 从推导方法可以看出  a>b  
&nbsp;&nbsp; 两者使用同向相加性  a-(1/a)>b-(1/b) 成立  
(4) |b|>|a| => ln|b|>ln|a|  
&nbsp;&nbsp; 运算规则:平方弄到前面去后面写绝对值  
&nbsp;&nbsp; 2ln|b|>2ln|a|  
&nbsp;&nbsp; lnb^2>lna^2  不成立  
答案为C  


###### ［例16］不等式组 [x-1≤a^2][x-4≥2a] 有解，则实数a的取值范围包含 ？个整数  
A `0`  
B `1`  
C `2`  
D `3`  
E `无穷多`  

先找到两个不等式区间有解就存在交集在找交集  
1.x≤a^2+1  
2.x≥2a+4  
![img_6.png](..%2F..%2Fpublic%2Fmath%2F3.%E6%95%B0%E5%AD%A6-%E4%B8%93%E9%A2%98%E5%BC%BA%E5%8C%96%E8%AF%BE%2Fimg_6.png)  
2a+4≤a^2+1

因式分解  
a 1  
a -3

a=3或a=−1  

数轴分析  
接下来我们利用这些解a=3和a=—1，将数轴分为三个区间：  
(1) a<-1  
(2) -1<a<3  
(3) a>3  

`a^2-2a-3≥0`各区间符号判断验证:    
(1) 当a<—1时：  
&nbsp;&nbsp; 比如选取a=—2来测试：  
&nbsp;&nbsp; (−2−3)(−2+1)=(−5)(−1)=5>0  
&nbsp;&nbsp; 所以在区间a<−1时，不等式a^2-2a-3≥0成立  
(2) 当-1<a<3时：

&nbsp;&nbsp; (0−3)(0+1)=(−3)(1)=-3<0

(3) 当a>3时：  
&nbsp;&nbsp; 比如选取a=4来测试：  
&nbsp;&nbsp; (4−3)(4+1)=(1)(5)=5>0  
&nbsp;&nbsp; 所以在区间a>3时，不等式成立

(1) 当a=-1时：  
&nbsp;&nbsp; (−1−3)(−1+1)=(−4)(0)=0  
&nbsp;&nbsp; 不等式0≥0成立  
(2) 当a=3时：  
&nbsp;&nbsp; (3−3)(3+1)=(0)(4)=0  
&nbsp;&nbsp; 不等式0≥0成立  

总上使用数轴分析推理出  
a≤-1 或 a≥3  
答案为E

# 一元二次不等式

## 1. 一元二次不等式的标准形式
ax^2+bx+c>0(或<0也存在≥跟≤的情况)  
其他非标准形式的不等式可以通过等价变形转化为标准形式

## 2. 解一元二次不等式的步骤
1.先化成标准型：ax^2+bx+c>0(或<0)，且a>0  
2.计算对应方程的判别式∆  
3.求对应方程根  
4.利用口诀“大于零在两边，小于零在中间”写出解集。   

## 3. 函数,方程,不等式的关系
常说的三个 `二次` 即指二次函数、一元二次方程和一元二次不等式，这三者之间有着密切的联系，
这种联系可以成为关键命题点，处理其中某类问题时，要产生对于另外两个`二次`的联想，或进行转化，
或帮助分析。具体到解一元二次不等式时，就是要善于利用相应的二次函数的图像进行解题分析，
要能抓住一元二次方程的根与一元二次不等式的解集区间的端点值的联系。

###### ［例17］不等式x^2-3x+2<0的解集是?
A `{x|x＜-2或x>-1}`      
B `{x|x＜1或x>2}`  
C `{x|1＜x<2}`  
D `{x|-2＜x<-1}`  
E `{x|-1<x<2}`  

十字相乘  
x -1  
x -2  
(x-1)(x-2)<0  
解不等式  
x-1=0 -> x=1  
x-2=0 -> x=2  
验证`(x-1)(x-2)<0`区间:    
x<1  
1<x<2  
x>2  

x<1: x=0   
(0-1)(0-2)=(-1)(-2)=2>0   
1<x<2: x=1.5   
(1.5-1)(1.5-2)=(0.5)(-0.5)=-0.25<0   
x>2: x=3   
(3-1)(3-2)=(2)(1)=2>0   

解集为: 1<x<2   
答案为: C   


###### ［例18］一元二次不等式3x^2-4ax+a^2<0(a<0)的解集是？  
A `a/3<x<a`  
B `x>a或a<a/3`  
C `a<x<a/3`  
D `x>a/3或x<a`  
E `a<x<3a`  

十字相乘  
1x -a  
3x -a  
(x-a)(3x-a)<0  
x-a=0 => x=a  
3x-a=0 => x=a/3  

验证`(x-a)(3x-a)<0`区间:    
其中a<0，所以a位于a/3的左边，因为a/3>a（当a<0时，负数被3除后会变得更大一些靠近0）  
x<a  
a<x<a/3  
x>a/3  

假设: a=-9  
当x<a => x<-9:  
x=-10  
(-10+9)(3(-10)+9)=(-1)(-21)=21>0  
当a<x<a/3 => -9<x<-3:     
x=-5  
(-5+9)(3(-5)+9)=(4)(-6)=-24<0  
当x>a/3 => x>-3:  
x=-2  
(-2+9)(3(-2)+9)=(7)(3)=21>0  

解集为: a<x<a/3  
答案为: C

###### ［例20］已知关于x的二次不等式ax^2+(a-1)x+a-1<0的解集为R，则a的取值范围是？
A `a<1/3`  
B `a>1/3`   
C `a<-(1/3)`  
D `a>-(1/3)`  
E `a>-3`  

列出恒成立的充分条件表达式：  
a<0  
∆=(a-1)^2-4a(a-1)<0  
这里两项中存在公因式不用展开完全平方公式直接用公因式求解:    
提取公因式(a−1)  
=> (a-1)[(a-1)-4a]  
=> (a−1)−4a = a−1−4a = -3a-1  
=> (a-1)(-3a-1)<0  
=> 提取负号: (a-1)(-(3a+1))<0  
=> 消除符号两边乘-1(不等式两边乘符号必须反转): (a-1)(3a+1)>0   
=> a=1  
=> a=-(1/3)  
验证`(a-1)(3a+1)>0`区间:  
a>1  
-(1/3)<a<1    
a<-(1/3)    

`当a>1 -> a=2:`  
(2-1)(3x2+1)=(1)(7)=7>0  
成立`但是恒成立的充分条件表达式已经明确表示a<0,所以负数减-1只会更负所以不成立`  
`当-(1/3)<a<1 -> a=-0.2:`  
(0.2-1)(3x0.2+1)=(-0.8)(1.6)=-1.28<0  
不成立  
`当a<-(1/3) -> a=-0.5`  
(-0.5-1)(3x-0.5+1)=(-1.5)(-0.5)=0.75>0  
成立  

解集为: a<-(1/3)   
答案为C


###### ［例21］已知(a^2-1)x^2-(a-1)x-1≥0的解集为空集，则实数a的取值范围中包含 ？个整数
A `0`  
B `1`  
C `2`  
D `3`  
E `无穷多`    

空集万能转换公式: f(x)≥0解为空集 <=> f(x)<0解为R(恒成立)   
(a^2-1)x^2-(a-1)x-1<0解集为R   
列出恒成立的充分条件表达式：  
a^2-1<0  
∆=(a-1)^2-4(a^2-1)<0  
解不等式组：  
1. a^2-1<0  
移项: a^2<1  
a的平方一定是小于1的所以只能0x0=0，然后0-1等于-1，得出a在-1到1之间  
-1<a<1  
2. (a-1)^2-4(a^2-1)<0  
提取(a^2-1)平方差公式: a^2-b^2=(a-b)(a+b)  
(a^2-1)=(a-1)(a+1)  
(a-1)^2-4(a-1)(a+1)<0  
提取公因式: (a-1)  
(a-1)^2 = (a-1)  
4(a-1)(a+1) = 4(a+1)  
(a-1)(4a+4) = (5a+3)  
(a-1)(5a+3)  

表达式简化后：  
-1<a<1  
(a-1)(5a+3)<0  
=> a=1  
=> a=-(3/5)  
验证`(a-1)(5a+3)<0`区间:  
a>1  
-(3/5)<a<1  
a<-(3/5)  

`当a>1 -> a=2:`  
(2-1)(5x2+3)=(1)(13)=13<0  
不成立  
`当-(3/5)<a<1 -> a=0:`  
(0-1)(5x0+3)=(-1)(3)=-3<0  
成立  
`当a<-(3/5) -> a=-1:`  
(-1-1)(5x-1+3)=(-2)(-2)=4<0  
不成立  

解集为: -(3/5)<a<1  
找出恒成立充分条件表达式的交集：  
从数轴上可以看到  
1. -(3/5)<a<1  
a的最小值为-(3/5)，最大值为 1  
3. -1<a<1  
a的最小值为-1，最大值为 1  

两个区间的重叠部分就为: `-(3/5)<a<1`  
这个范围内的整数是 0 只有1个整数

判断到这里答案范围内的整数可能会出错:   
因为题目没有写必须要求是这个函数是二次不等式所以a有可能为0  
当a^2-1=0时： `a=+-1 `  
根据空集万能转换公式需要满足全体实数解集为R(才成立该表达式)  
负无穷<x(0)<正无穷 = R(恒成立)  
解一元一次表达式：  
1. `当a=-1时:`  
((-1)^2-1)x^2-((-1)-1)x-1<0  
=> 0+2x-1<0  
=> 2x-1<0  
=> x<1/2  
不成立 只满足x=(负无穷，1/2)这个区间的值
2. `当a=1时:`
((1)^2-1)x^2-(1-1)x-1<0  
=> 0-0-1<0  
=> -1<0  
成立 `−1<0`是一个真命题成立的不等式表示无论一元一次x取什么值解都为`−1<0`

综上  
`一元二次不等式` a的范围在 `-(3/5)<a<1`   
这个范围内的整数是 `0` 只有 `1` 个整数  
`一元一次不等式` a的范围在 `a=1`  
这个范围内的整数是 `1` 只有 `1` 个整数  
答案为B



###### ［例22］若不等式2x^2+2kx+k/4x^2+6x+3 < 1对于一切实数x都成立，则实数k的范围中包含 ? 个整数
A `0`  
B `1`  
C `2`  
D `3`  
E `无穷多`


分母：4x^2+6x+3 全部是正数`抛物线为恒正`开口向上  
∆= 6^2-4x4x3<0  
为了消去分母，可以将不等式两边同时乘以 4x^2+6x+3  
`注意：做这个操作时，我们需要确保分母是正的，因为如果分母为负，那么不等式的方向会改变`  
2x^2+2kx+k < 4x^2+6x+3  
移项：  
=> 0 < 4x^2+6x+3-2x^2-2kx-k  
合并同类项:  
二次项：  
=> 4x^2-2x^2 = 2x^2  
一次项:  
=> 6x-2kx = (6-2k)x  
常数项：  
=> 3-k  
结合加起来:  
2x^2+(6-2k)x+3-k > 0  (恒成立符合`抛物线为恒正`开口向上)  
∆=(6-2k)^2-8(3-k)<0   
求∆根：  
展开`(6-2k)^2`平方公式：(a-b)^2 = a^2-2ab+b^2  
(6-2k)^2 = 36-24k+4k^2  
展开`-8(3-k)`  
-8(3-k) = -24+8k  
将这两部分合并：  
36-24k+4k^2-24+8k < 0  
合并同类项：  
4k^2-16k+12 < 0 

采用两种方式解法：  
十字相乘法：  
适用于可以通过分解来找到整数根或分数根的方程   
不能解根为非整数或分数(例如虚数根)  
不能解超过最高次项为2或不易分解的多项式    
求根公式：  
适用于所有一元二次方程,不论系数的大小或复杂性  
可以解根为非整数或分数(例如虚数根)  
可以解超过最高次项为2或不易分解的多项式  
需要最简化表达式不然会导致平方根过大,计算过程复杂难处理  
根的性质依赖于判别式  
![img_12.png](..%2F..%2Fpublic%2Fmath%2F3.%E6%95%B0%E5%AD%A6-%E4%B8%93%E9%A2%98%E5%BC%BA%E5%8C%96%E8%AF%BE%2Fimg_12.png)  


1. 求根公式（`必须要最简化成最简的表达式要不然平方根太大求不出根`）  
公因式分解提取每项中的最大公因数:  
4(k^2−4k+3)  
最大公因数为4  
4k^2-16k+12除4 = k^2−4k+3 < 0  
找到不等式的根转换求解相应的方程：  
k^2−4k+3 = 0  
![img_11.png](..%2F..%2Fpublic%2Fmath%2F3.%E6%95%B0%E5%AD%A6-%E4%B8%93%E9%A2%98%E5%BC%BA%E5%8C%96%E8%AF%BE%2Fimg_11.png)  
k=3  
k=1

2. 十字相乘  
简化表达式便于后面   
4k^2-16k+12 < 0    
4k &nbsp; -4   
k &nbsp; -3  
(4k-4)(k-3)<0   
4k-4=0 => k=1   
k-3=0 => k=3  

验证区间:  
k>3  
1<k<3  
k<1  

验证区间最好使用联想抛物线图像进行推理:   
1. 十字相乘`(4k-4)(k-3)<0`验证推理    
`当k>3 -> k=4:`  
(4x4-4)(4-3)=(12)(1)=12<0  
不成立  
`当1<k<3 -> k=2:`  
(4x2-4)(2-3)=(4)(-1)=-4<0    
成立  
`当k<1 -> k=0:`  
(4x0-4)(0-3)=(-4)(-3)=12<0  
不成立  
2. 求根公式`k^2−4k+3 < 0`验证推理  
`当k>3 -> k=4:`  
4^2−4x4+3=16-16+3=3<0
