## 习题
```
〖例1〗
  ①|m³-3|=1 => m²-3=1 或 m²-3=-1
                  ↓         ↓
                m²-3=1     m²=2
                  ↓         ↓
                m=±2       m=±√2
 ②m-2≠0 => m≠2 => m=-2,±√2 
 

〖例2〗
  ①-②: 5𝑦-(-𝑦)=-3k-9k
             6𝑦=-12k
              𝑦=-2k
  代入②: 𝑥-(-2k)=9k 
         => 𝑥=9k-2k 
         => 𝑥=7k
  𝑥=7k        
  𝑦=-2k 
  代入③: 2x7k+3x(-2k)=6
         => 14k-6k=6
         => 8k=6
         => k=6/8
         => 3/4
    
         
〖例3〗             
  因式分解十字相乘主要看系数
    2a²-13a+15
    => 1⬁⬀-5
       2⬃⇘-3
    => -10-3 = -13
    => (a-5)(2a-3)
    
    a²𝑥²-(3a²-8a)𝑥+2a²-13a+15=0
    -(3a²-8a)
    => -(3a-8)
     => a⬁⬀-(a-5)
        a⬃⇘-(2a-3)
     => -a(2a-3)-a(a-5)
     => -a(2a-3+a-5)
     => -a(3a-8)
     => [a𝑥-(a-5)][a𝑥-(2a-3)]
     
     a𝑥=a-5          a𝑥=2a-3
   𝑥1=> a-5/a      𝑥2=> 2a-3/a 
     => 1-(5/a)      => 2-(3/a)
     => a= 5/a       => a= 3/a
     
     5÷1=5            3÷1=3  
     5÷5=1            3÷3=1
                                 
     a=1,3,5
     

〖例4〗 
  求根公式: a𝑥²+b𝑥+c=0
  韦达定理: 𝑥1+𝑥2=-(b/a)
           𝑥1𝑥2=c/a 
  甲: c错误 a,b正确 𝑥1+𝑥2=-(b/a)
                   => 8+2
                   => 10
  乙: b错误 a,c正确 𝑥1𝑥2=c/a
                   => -9x-1
                   => 9
  a(𝑥-≈1)(𝑥-𝑥2)=0
  乘法分配公式: a(b+c)=ab+ac
  => 𝑥²-(𝑥1+𝑥2)𝑥+𝑥1𝑥2=0                 
  => 𝑥²-10𝑥+9=0
  => (𝑥-1)(𝑥-9)=0
  => 𝑥1=1  𝑥2=9
  
  
〖例5〗
  𝑥1+𝑥2=-(p/1)=-p
  𝑥1𝑥2=37/1=37
  𝑥1=1
  𝑥2=37
  => p=-(𝑥1+𝑥2)=-38
  => (𝑥1+1)(𝑥2+1)/p
  => 2x38/-38
  => -2
```
