###### 〖例4〗
求根公式: a𝑥²+b𝑥+c=0  
韦达定理: 𝑥1+𝑥2=-(b/a)  
𝑥1𝑥2=c/a  
甲: c错误 a,b正确 𝑥1+𝑥2=-(b/a)  
=> 8+2  
=> 10  
乙: b错误 a,c正确 𝑥1𝑥2=c/a  
=> -9x-1  
=> 9  
a(𝑥-≈1)(𝑥-𝑥2)=0  
乘法分配公式: a(b+c)=ab+ac  
=> 𝑥²-(𝑥1+𝑥2)𝑥+𝑥1𝑥2=0                   
=> 𝑥²-10𝑥+9=0  
=> (𝑥-1)(𝑥-9)=0  
=> 𝑥1=1  𝑥2=9
