###### 〖例1〗若a,b,c满足|a-3| + √3b+5 + (5c-4)² = 0 则abc=( )

###### 〖例2〗|𝑥-1|+|𝑥-3| = 4-2𝑥 其非负整数解有( )个
分界点: |𝑥-1|=1  |𝑥-3|=3  [1,3]  
① 当 𝑥≤1 时  𝑥-1≤0  𝑥-3≤1-3≤0  
=> 1-𝑥+3-𝑥 = 4-2𝑥  
=> 4-2𝑥 = 4-2𝑥 恒等式成立  
= |𝑥≤1|  
=> 0 || 1  
② 当 1<𝑥<3 时  𝑥-1>0  𝑥-3<0  
=> 𝑥-1+3-𝑥 = 4-2𝑥  
=> 2 = 4-2𝑥  
=> 𝑥 = 1  
不满足 1<𝑥<3 条件  
③ 当 𝑥≥3 时  𝑥-1>0  𝑥-3>0  
=> 𝑥-1+𝑥-3 = 4-2𝑥  
=> 2𝑥-4 = 4-2𝑥  
=> 4𝑥 = 8  
=> 𝑥 = 2  
不满足 𝑥≥3 条件  

###### 〖例3〗(条件充分性判断) b+c/|a| + c+a/|b| + b+a/|c| = 1
(1) 实数 a,b,c 满足 a+b+c=0  
(2) 实数 a,b,c 满足 abc>0  
A.(1)✔︎ (2)✖︎  
B.(1)✖︎ (2)✔︎  
D.(1)✔︎ (2)✔︎  
联合:  
C.(1)x(2)✖︎ (1)+(2)✔︎  
E.(1)x(2)✖︎ (1)+(2)✖︎  

条件(1) = b+c/|a| + c+a/|b| + b+a/|c| = -a/|a| + -b/|b| + -c/|c|  
无法知道 a,b,c 的数值 ✖︎  
条件(2) 无法知道 a,b,c 的数值 ✖︎  
联合:  
(2)abc>0  
(1)-a/|a| + -b/|b| + -c/|c|  
= -(a/|a| + b/|b| + c/|c|)

a+b+c=0  
abc>0  
如果a,b,c都大于0 那a+b+c就不可能等于0  
所以一定有负数 根据负负得正规则  
所以存在两负一正  
假设 a,b=-1 c=1    
=> -(-1 + -1 + 1)  
=> 1
