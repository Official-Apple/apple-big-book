###### 〖例2〗不等式𝑥(6𝑥+5)<4
(1) 𝑥>-1  
(2) 𝑥<1/3  

6𝑥²+5𝑥 < 4  
=> 6𝑥²+5𝑥-4 < 0  
因式分解十字相乘法:  
𝑥  常数  
2  -1  
3  4  
2x4+3x(-1)  
= 8-3  
= 5  
=> (2𝑥-1)(3𝑥+4) < 0   
=> -4/3 < 𝑥 < 1/2  
联合(1)(2)条件推理:  
-4/3  -1  𝑥  1/3  1/2    
(1)+(2)✔︎
