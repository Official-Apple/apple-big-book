3.基本公式    
⑷ 立方和(差)公式
###### 〖例5〗已知a-b=2,a³-b³=28,则a²+b²=( )
a³-b³=(a-b)(a²+b²+ab)       
=> a²+b²+ab = 28/2 = 14 ①  
=> (a²-b²) = a²+b²-2ab = 2² = 4 ②  
消元法:  
① x 2: 2(a²+b²)+2ab = 28 ③  
② + ③: 3(a²+b²) = 32  
=> (a²+b²) = 32/3
